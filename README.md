# brontes

> *brontes* (Greek: thunder). In myth, the Cyclops smith who forged Zeus's thunderbolts. This crate will forge clap CLIs into MCP servers.

A Rust library for transforming `clap` CLIs into [MCP](https://modelcontextprotocol.io) servers, inspired by [njayp/ophis](https://github.com/njayp/ophis).

## Status

Phase 0 scaffold. The public API is not yet stabilized and no functionality is exposed at this revision. The crate currently builds, lints, and ships an internal error type only.

Tracking issue, design notes, and the phased rollout plan live in the project repository.
